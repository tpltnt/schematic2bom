About
=====

a simple Ruby script to convert KiCAD schematics to BOMs (in CSV)

Usage
-----
*Be warned, this script is **very** optimistic about the running conditions.*
simply run ``./schematic2bom.rb <SCHEMATICFILENAME>``
the field seperator is ";"
